# Quarterly Portfolio Review

This file contains the process and checklist for quarterly investment reviews.

---

## Review Schedule

| Quarter | Review Window | Focus |
|---------|---------------|-------|
| Q1 | Early April | Full-year earnings, annual guidance |
| Q2 | Early July | Mid-year check, adjust for H2 |
| Q3 | Early October | Q3 earnings, holiday guidance |
| Q4 | Early January | Year-end review, tax considerations, annual planning |

**Reminder**: Set calendar reminders for the first week of April, July, October, January.

---

## Pre-Review Preparation

Before starting the review, gather:

1. **Current portfolio positions and allocations**
2. **Recent earnings reports** for each holding (last quarter)
3. **Hyperscaler capex guidance** (MSFT, GOOGL, AMZN, META earnings calls)
4. **Macro indicators**:
   - AI application revenue growth (if available)
   - Semiconductor demand signals
   - Interest rates / credit conditions

### Speculative Positions (POET, ONDS)

For speculative positions, the bar is simpler:

- [ ] **Is the thesis still alive?** (Not dead yet)
- [ ] **Cash runway >12 months?**
- [ ] **Any major validation events?** (Contracts, partnerships, revenue)
- [ ] **Any thesis-killing events?** (Tech failure, key partner loss)

**Action**: [ ] Hold / [ ] Add / [ ] Sell / [ ] Let ride to zero

---

## Red Flags That Require Immediate Action

Don't wait for quarterly review if these happen:

1. **Contract cancellation** for IREN or CIFR → Review immediately
2. **Hyperscaler capex cut >20%** → Review all AI infrastructure positions
3. **Management departure** (CEO/CFO) → Review that position
4. **Fraud/accounting scandal** → Sell immediately
5. **Stock drops >40% on news** → Review whether thesis still valid
